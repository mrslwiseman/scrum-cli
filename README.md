# Usage
<!-- usage -->
```sh-session
$ npm install -g scrum
$ scrum COMMAND

$ scrum (-v|--version|version)
scrum/0.0.0 darwin-x64 node-v10.11.0
$ scrum --help [COMMAND]
USAGE
  $ scrum COMMAND
...
```
